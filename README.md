## 🛰️ Sobre o Projeto

Este site foi desenvolvido para o concurso **Agrinho**, com foco em apresentar a convergência entre o **agronegócio produtivo** e a **sustentabilidade**. Através de uma interface tecnológica e futurista, o usuário explora como a inovação (Agro 4.0) é a chave para o equilíbrio ecológico.

### 🎯 Objetivos:
* **Educação Ambiental:** Mostrar práticas de preservação de nascentes e solo.
* **Tecnologia Aplicada:** Exibir o uso de drones, sensores de solo e IA.
* **Conscientização:** Provar que "Agro Forte" e "Natureza Viva" são conceitos inseparáveis.

## 🚀 Funcionalidades Complexas

1.  **Interface Dinâmica:** O cabeçalho muda de transparência conforme o usuário rola a página, garantindo legibilidade.
2.  **Counter Logic:** As estatísticas de sustentabilidade não são estáticas; elas "contam" de 0 até o valor final quando entram na tela.
3.  **Grid Adaptativo:** O layout se reorganiza automaticamente para qualquer tamanho de tela (Mobile First).
4.  **Ecossistema Visual:** Uso de filtros de fundo (`backdrop-filter`) para criar uma estética "High-Tech" limpa.
