# Agro Forte - Futuro Sustentável

Este é um site informativo dedicado a promover o uso de práticas agrícolas sustentáveis e as inovações tecnológicas que estão moldando o futuro da agricultura. O site oferece informações sobre o impacto da sustentabilidade no agro, tecnologia no campo, e práticas responsáveis para um futuro mais verde.

## Funcionalidades

- **Modo Escuro/Claro**: Alterna entre temas claro e escuro com base na preferência do usuário. O estado é salvo no navegador para manter a preferência mesmo após recarregar a página.
- **Carregamento Dinâmico de Vídeos**: Vídeos educativos sobre temas como agricultura sustentável, tecnologia no agro, e o futuro da agricultura são carregados dinamicamente na seção de vídeos.
- **Interatividade na FAQ**: A seção de perguntas frequentes (FAQ) permite que o usuário clique nas perguntas para expandir e ver as respostas.
- **Design Responsivo**: O site é totalmente responsivo e se adapta a diferentes tamanhos de tela, sendo adequado para dispositivos móveis e desktops.

## Tecnologias Usadas

- **HTML5**: Estrutura da página e conteúdo semântico.
- **CSS3**: Estilos e design, incluindo efeitos de transição e responsividade.
- **JavaScript**: Funcionalidades dinâmicas, como alternância de temas e carregamento de vídeos.
- **YouTube**: Integração de vídeos usando `iframe` para exibição de conteúdo educacional.
