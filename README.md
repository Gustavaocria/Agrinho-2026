# Agro Forte - Futuro Sustentável

## Descrição
Este projeto é um site informativo inspirado em portais oficiais do Paraná, com foco em agricultura sustentável e equilíbrio ambiental. O site é moderno, responsivo, chamativo e possui interatividade.

## Funcionalidades
- **Modo Escuro**: botão para alternar entre modo claro e escuro
- **Contador de visitas**: registra cliques do usuário
- **Mostrar/Ocultar Dicas**: dicas de agricultura sustentável aparecem ao clicar
- **Alterar slogan**: botão que muda dinamicamente o texto do slogan
- **Formulário de contato**: envia mensagem com validação básica
- **Seções informativas**: Sobre, Projetos, Notícias, Links úteis
- **Responsivo**: funciona em desktop, tablet e mobile

## Tecnologias usadas
- **HTML5**: estrutura semântica do site
- **CSS3**: estilos, animações e modo escuro
- **JavaScript**: manipulação do DOM e interatividade
