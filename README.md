# 🌱 Agro Forte - Agrinho Paraná 2026

> 🌾 Tema: Agro Forte: Futuro Sustentável – equilíbrio entre produção e meio ambiente

---

## 🚀 Sobre o Projeto

O Agro Forte é um site interativo desenvolvido para o programa Agrinho Paraná, com o objetivo de conscientizar sobre práticas agrícolas sustentáveis e a preservação do meio ambiente.

O projeto vai além de um site informativo, oferecendo interação direta com o usuário, permitindo que ele calcule seu impacto ambiental e reflita sobre seus hábitos.

---

## 🧠 Funcionalidades

✅ Sistema de login (simulado)  
✅ Modo escuro 🌙  
✅ Calculadora ambiental interativa 🌍  
✅ Barra de impacto em tempo real 📊  
✅ Vídeo educativo incorporado 🎬  
✅ Área de contato (reclamações/sugestões) 📩  
✅ Design moderno com cores da natureza 🌿  
✅ Manipulação do DOM com JavaScript  

---

## 🎯 Objetivo

Promover a conscientização sobre:

- Sustentabilidade no campo 🌱  
- Uso consciente de recursos naturais 💧  
- Redução de impactos ambientais 🌍  
- Produção agrícola responsável 🌾
