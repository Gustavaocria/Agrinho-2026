## 🚀 Sobre o Projeto

O Agro Forte é um site interativo desenvolvido para o programa Agrinho Paraná 2026, com foco em promover a conscientização sobre sustentabilidade no campo.

💡 O grande diferencial do projeto é permitir que o usuário interaja com o conteúdo, entendendo seu impacto ambiental de forma prática.

---

## ✨ Destaques do Projeto

🌱 Design moderno com cores da natureza  
🌙 Interface agradável e organizada  
🎬 Vídeo educativo integrado  
🧠 Calculadora ambiental interativa  
📊 Barra de impacto em tempo real  
📩 Sistema de contato  
🔐 Tela de login  
💻 Estrutura profissional com múltiplas páginas  

---

## 🧠 Funcionalidades

- Manipulação do DOM com JavaScript  
- Interação em tempo real com o usuário  
- Navegação entre páginas (estrutura profissional)  
- Simulação de sistema de login  
- Feedback visual dinâmico  

---

## 🎯 Objetivo

Promover a conscientização sobre:

- 🌍 Sustentabilidade  
- 💧 Uso consciente da água  
- 🌾 Produção agrícola responsável  
- ♻️ Redução de impactos ambientais
