## 🚀 Sobre o Projeto

O Agro Forte é um site interativo desenvolvido para o Agrinho 2026 com foco em sustentabilidade no campo.

💡 O diferencial é a calculadora ambiental, que permite ao usuário entender seu impacto no meio ambiente.

---

## ✨ Funcionalidades

- 🔐 Login (simulado)
- 🌙 Modo escuro
- 🧠 Calculadora ambiental
- 📊 Barra de impacto
- 🎬 Vídeo educativo
- 📩 Formulário de contato
- 💻 Design moderno e responsivo

---

## 🎯 Objetivo

Promover a conscientização sobre:

- 🌱 Sustentabilidade

- 🌾 Produção responsável
- ♻️ Preservação ambiental
