# Agro Forte - Futuro Sustentável

Agro Forte é um site informativo e moderno que promove o equilíbrio entre **produção agrícola e preservação ambiental**, com foco em tecnologia e sustentabilidade.

## 🌱 Funcionalidades

O site foi desenvolvido utilizando **HTML, CSS e JavaScript**, oferecendo um design limpo, responsivo e interativo.

### 1. Estrutura do Site
- **Header fixo** com menu de navegação e botão de modo escuro.
- **Seções principais**:
  - Sobre o Agro Forte
  - Galeria de fotos interativa
  - Vídeos educativos incorporados
  - Últimas notícias do setor agro
  - Links úteis externos

### 2. Funcionalidades em JavaScript (Manipulação do DOM)
O JavaScript do site **interage diretamente com o DOM** para tornar a experiência do usuário dinâmica:

1. **Modo Escuro**
   - Ao clicar no botão "Modo Escuro", a aparência do site muda.
   - O texto do botão é alterado dinamicamente para indicar o estado atual (`Modo Claro` ou `Modo Escuro`).

2. **Mostrar/Esconder Galeria**
   - Um botão acima da galeria permite **ocultar ou mostrar as fotos** sem recarregar a página.

3. **Contador de Cliques nos Cards**
   - Cada card da seção "Sobre" possui um **contador de cliques**.
   - O número de interações é exibido na tela em tempo real, demonstrando manipulação de elementos do DOM.

4. **Alteração de Texto Dinâmica**
   - O título principal da página é atualizado automaticamente após alguns segundos, mostrando como o JS pode alterar o conteúdo do HTML.

5. **Scroll Suave**
   - O menu de navegação permite rolar a página de forma suave até cada seção.

### 3. Tecnologias Utilizadas
- **HTML5** – Estrutura semântica do site
- **CSS3** – Estilização moderna, responsiva e modo escuro
- **JavaScript** – Interatividade e manipulação do DOM
- **Git/GitHub** – Controle de versão e publicação do site
